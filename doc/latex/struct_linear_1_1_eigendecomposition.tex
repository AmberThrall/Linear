\doxysubsection{Detailed Description}
\subsubsection*{template$<$typename T, size\+\_\+t N, unsigned int Flags = 0$>$\newline
struct Linear\+::\+Eigendecomposition$<$ T, N, Flags $>$}

Struct for \mbox{\hyperlink{struct_linear_1_1_eigendecomposition}{Eigendecomposition}}. 

This struct finds NxN matrix Q and NxN matrix D such that $A=QDQ^{-1}$, D is diagonal with entries equaling the eigenvalues of A and V\textquotesingle{}s columns are the corresponding eigenvectors of A. Only diagonalizable matrices can be decomposed in this fashion. 
\begin{DoxyParams}{Parameters}
{\em T} & Type to store matrix entries as. \\
\hline
{\em N} & Number of rows/columns for L and D (both square). Dynamic is allowed for N. \\
\hline
{\em Flags} & Flags to pass to the matrices (default = row major). \\
\hline
\end{DoxyParams}

\subsubsection*{template$<$typename T, size\+\_\+t N, unsigned int Flags = 0$>$\newline
struct Linear\+::\+Hessenberg$<$ T, N, Flags $>$}

Struct for \mbox{\hyperlink{struct_linear_1_1_hessenberg}{Hessenberg}} decomposition. 

This struct NxN matrix Q and NxN matrix H such that $A=QHQ^*$ where Q is unitary and H is upper \mbox{\hyperlink{struct_linear_1_1_hessenberg}{Hessenberg}}. 
\begin{DoxyParams}{Parameters}
{\em T} & Type to store matrix entries as. \\
\hline
{\em N} & Number of rows/columns for Q and H (both square). Dynamic is allowed for N. \\
\hline
{\em Flags} & Flags to pass to the matrices (default = row major). \\

\end{DoxyParams}

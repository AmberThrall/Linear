\subsubsection*{template$<$typename T, size\+\_\+t N, unsigned int Flags = 0$>$\newline
struct Linear\+::\+Schur$<$ T, N, Flags $>$}

Struct for \mbox{\hyperlink{struct_linear_1_1_schur}{Schur}} decomposition. 

This struct NxN matrix Q and NxN matrix U such that $A=QUQ^*$ where Q is unitary and U is block upper triangular with block sizes 1x1 and 2x2. 
\begin{DoxyParams}{Parameters}
{\em T} & Type to store matrix entries as. \\
\hline
{\em N} & Number of rows/columns for Q and U (both square). Dynamic is allowed for N. \\
\hline
{\em Flags} & Flags to pass to the matrices (default = row major). \\
\hline
\end{DoxyParams}

\subsubsection*{template$<$typename T, size\+\_\+t M, size\+\_\+t N, unsigned int Flags = 0, SVDType Type = FULL\+\_\+\+SVD$>$\newline
struct Linear\+::\+SVD$<$ T, M, N, Flags, Type $>$}

Struct for \mbox{\hyperlink{struct_linear_1_1_s_v_d}{SVD}} decomposition. 

This struct finds MxM matrix U, MxN matrix S and NxN matrix V such that $A=USV^*$ where U and V are both unitary, and S is a rectangular diagonal matrix containing the real non-\/negative singular values of A in decreasing order. If THIN\+\_\+\+SVD is passed, then U is MxK, S is KxK, and Vh is KxN where K=min(\+M,\+N). 
\begin{DoxyParams}{Parameters}
{\em T} & Type to store matrix entries as. \\
\hline
{\em M} & Number of rows for U and S. Dynamic is allowed for M. \\
\hline
{\em N} & Number of columns for S and Vh. Dynamic is allowed for N. \\
\hline
{\em Flags} & Flags to pass to the matrices (default = row major). \\
\hline
{\em Type} & Either FULL\+\_\+\+SVD or THIN\+\_\+\+SVD (default = FULL\+\_\+\+SVD). \\
\hline
\end{DoxyParams}

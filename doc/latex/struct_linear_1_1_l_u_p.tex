\doxysubsection{Detailed Description}
\subsubsection*{template$<$typename T, size\+\_\+t N, unsigned int Flags = 0$>$\newline
struct Linear\+::\+LUP$<$ T, N, Flags $>$}

Struct for \mbox{\hyperlink{struct_linear_1_1_l_u_p}{LUP}} decomposition. 

This struct finds NxN matrix L, NxN matrix U and NxN matrix P such that $PA=LU$, L is lower triangular, U is upper triangular and P is a permutation matrix. 
\begin{DoxyParams}{Parameters}
{\em T} & Type to store matrix entries as. \\
\hline
{\em N} & Size of the L,U,P matrices (all three are square). Dynamic is allowed for N. \\
\hline
{\em Flags} & Flags to pass to the matrices (default = row major). \\
\hline
\end{DoxyParams}

\subsubsection*{template$<$typename T, size\+\_\+t M, size\+\_\+t N, unsigned int Flags = 0$>$\newline
struct Linear\+::\+QR$<$ T, M, N, Flags $>$}

Struct for \mbox{\hyperlink{struct_linear_1_1_q_r}{QR}} decomposition. 

This struct finds MxN matrix Q, NxN matrix R such that $A=QR$ and R is upper triangular. Moreover, if Q is square then Q is unitary. 
\begin{DoxyParams}{Parameters}
{\em T} & Type to store matrix entries as. \\
\hline
{\em M} & Number of rows for Q. Dynamic is allowed for M. \\
\hline
{\em N} & Number of columns for Q and R (R is square). Dynamic is allowed for N. \\
\hline
{\em Flags} & Flags to pass to the matrices (default = row major). \\
\hline
\end{DoxyParams}

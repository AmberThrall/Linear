\doxysubsection{Detailed Description}
\subsubsection*{template$<$typename T, size\+\_\+t N, unsigned int Flags = 0$>$\newline
struct Linear\+::\+Cholesky$<$ T, N, Flags $>$}

Struct for \mbox{\hyperlink{struct_linear_1_1_cholesky}{Cholesky}} decomposition. 

This struct finds NxN matrix L and NxN matrix D such that $A=LDL^*$, L is lower unit triangular and D is diagonal. 
\begin{DoxyParams}{Parameters}
{\em T} & Type to store matrix entries as. \\
\hline
{\em N} & Number of rows/columns for L and D (both square). Dynamic is allowed for N. \\
\hline
{\em Flags} & Flags to pass to the matrices (default = row major). \\
\hline
\end{DoxyParams}
